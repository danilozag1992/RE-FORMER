## Usage

> Clone the repository to your local machine

```sh
$ git clone https://github.com/danilozag1992/RE-FORMER.git
```
> cd into the directory

```sh
$ cd RE-FORMER
```
##### To create Database
```sh
$ rails db:create
```
##### Run Migrations
```sh
$ rails db:migrate
```
##### Install Gems
```sh
$ bundle install
```
##### Run Server
```sh
$ rails s
```
